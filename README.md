# Punto Pastelero

Landing page para sistema de visibilidad compartida para negocios de comida.

## Instrucciones

1. Instalar dependencias:
```bash
npm install
```

2. Ejecutar en desarrollo:
```bash
npm run dev
```

3. Abrir [http://localhost:3000](http://localhost:3000)

## Configuración

- Cambiar el número de WhatsApp en `app/page.tsx` (línea 4)
- Los estilos están configurados con Tailwind CSS
